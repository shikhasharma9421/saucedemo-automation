# SauceDemo Automation Framework (Selenium + Java + TestNG)

This is my first automation testing project, and I am building the framework structure myself. The framework is designed using **Page Object Model (POM)** to keep tests clean, reusable, and maintainable.

---

## Tech Stack
- Java  
- Selenium WebDriver  
- TestNG  
- Maven  
- IntelliJ IDEA  

---

## Application Under Test

Website: [SauceDemo](https://www.saucedemo.com/)

This project automates the SauceDemo web application, covering the login functionality and the end-to-end checkout flow.

---

## Folder Explanation

**base/**  
Common setup and teardown methods, such as browser initialisation and driver management.

**pages/**  
Contains Page Object Model classes where web elements and actions for each page are defined.

**tests/**  
Contains TestNG test classes that execute the test scenarios using the page classes.

---

## Test Scenarios Covered

- Login functionality  
- Inventory page validations  
- Add product to cart  
- End-to-end checkout flow  
  - Login  
  - Add items to cart  
  - Open cart  
  - Checkout process  

---

## Project Goal

The goal of this project is to practice building a **scalable Selenium automation framework from scratch**, following industry best practices like **Page Object Model and clean project structure**.
